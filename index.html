🎮 NES Emulator — JS Puro v14

Emulador próprio, sem EmulatorJS e sem core externo. Suporta leitura de ROM .nes iNES, Mapper 0/NROM e Mapper 4/MMC3 experimental. CPU 6502, PPU v14 inteligente por ROM, base v8 para MMC3 e correções específicas para Mario. Esta versão usa a base gráfica estável da v8 e aplica correções inteligentes por perfil de ROM.

Status: aguardando ROM .nes

Controles: Setas = D-Pad | Z = A | X = B | Enter = Start | Shift = Select
